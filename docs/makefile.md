# Makefile qo'llanma

Foydali targetlar:

- `make build` — Docker image(lar)ni qurish
- `make up` — xizmatlarni ko'tarish (-d)
- `make down` — to'xtatish
- `make restart` — Django konteynerini qayta ishga tushirish
- `make logs` — loglarni ko'rish
- `make shell` — Django shell
- `make migrate` — migratsiyalar
- `make createsuperuser` — admin yaratish
- `make collectstatic` — statik fayllar
- `make test` — testlar
- `make setwebhook` — Telegram webhook o'rnatish
- `make deletewebhook` — webhookni o'chirish
- `make webhookinfo` — joriy webhook ma'lumoti
